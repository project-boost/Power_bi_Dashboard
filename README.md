E-Commerce Delivery Performance Dashboard (Power BI)
📌 Project Overview

This project presents an interactive Power BI dashboard designed to analyze delivery performance in an e-commerce environment. The dashboard helps visualize key operational metrics such as average delivery time, customer ratings, order accuracy, payment methods, and discount usage to gain insights into delivery efficiency and customer satisfaction.

The goal of this project is to transform raw delivery data into meaningful insights using data visualization and business intelligence techniques.

🎯 Project Objectives

Analyze delivery performance across different agents.

Evaluate customer satisfaction using rating metrics.

Monitor order accuracy and delivery time.

Understand payment mode distribution and discount usage.

Provide interactive insights through dashboards.

📂 Dataset

The dataset used in this project contains information related to e-commerce delivery operations, including:

Delivery Agent Name
Delivery Time
Customer Rating
Customer Service Rating
Order Accuracy
Payment Mode (UPI, Credit Card, Debit Card, COD)
Order Type (Online / In-Store)
Discount Applied
Delivery Location / City

The dataset was cleaned and structured before visualization.

⚙️ Tools & Technologies
Power BI – Data visualization and dashboard creation
Microsoft Excel / Dataset Source – Data preparation
Data Visualization Techniques – Charts, KPIs, slicers

📊 Dashboard Features

KPI Cards
Average Rating
Customer Service Rating
Order Accuracy Count
Average Delivery Time
Agent Performance Analysis
Delivery time comparison across agents
Customer rating comparison
Location-Based Insights
Customer service rating by city
Discount distribution by location
Payment Mode Analysis
Distribution of payment methods used by customers
Interactive Filters
Agent Name
Payment Mode
Order Type

These filters allow users to dynamically explore the data.
